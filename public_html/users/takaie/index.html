研究テーマ

テーマ
コンテナにおけるライブマイグレーション
概要:
Docker を代表とするコンテナ型の仮想化技術が普及している．Docker を用いてソフトウェアサービスを構築する場合，ネットワーク上に分散した複数の計算機上に，サービスの機能単位でコンテナと呼ばれる隔離された空間を作成する． そして，これらのコンテナの連携でサービスを構築することが多い． したがって，各コンテナに配置するプロセスの種類と数，および各計算機に配置するコンテナの種類と数を設計で適切に決定することが重要になる． サービス環境を適切に設計するためには，コンテナ間の通信のオーバヘッドによるサービス性能の低下を考慮する必要がある．構築後に設計を変更する場合には煩雑な作業が必要となる．
そこで，ライブマイグレーションという技術を利用し，コンテナ内で走行しているプロセスを別のコンテナに移行するという方法を考える．これにより，構成を容易に変更できるため，実際にサービス境を構築し，プロセスまたはコンテナの配置を試行錯誤することで最適な配置を検討できると考える．
